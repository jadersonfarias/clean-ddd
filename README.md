# Subdomínios 

- Core: o que dá dinheiro
- Supporting : Dá suporte para o core funcionar
- Generic: Você precisa, mas não são tão importantes

## Exemplos 


### Core

- Compra
- Catálogo
- Pagamento
- Entrega

### Suporting

- Estoque

### Generic

- Notificação ao cliente
- Promoções
- chat
